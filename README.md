# snMultiome_EOAD
Single-nucleus multiomics reveals the disrupted regulatory programs in three brain regions in sporadic early-onset Alzheimer's disease

# Description
Sporadic early-onset Alzheimer’s disease (sEOAD) represents a significant but less-studied subtype of Alzheimer’s disease (AD). Here, we generated a single-nucleus multiome atlas derived from the postmortem prefrontal cortex, entorhinal cortex, and hippocampus of nine individuals with or without sEOAD. The scripts contain code for generating main figures in the current version of the manuscript.
